# Retail Sales Analytics Platform

## Overview

Retail Sales Analytics Platform is an end-to-end business analytics solution designed to automate sales reporting, inventory analysis, stock coverage monitoring, revenue forecasting, and executive summary generation.

The project simulates a real-world retail analytics environment and demonstrates how modern data analytics pipelines can transform raw transactional data into actionable business insights.

---

## Key Features

### Sales Analytics

* Top Selling Products Analysis
* Low Selling Products Analysis
* Revenue Performance Tracking
* Product-wise Performance Metrics

### Inventory Analytics

* Warehouse Stock Monitoring
* Stock Coverage Calculation
* Inventory Health Analysis
* Replenishment Indicators

### Forecasting Engine

* Revenue Forecasting
* Product-Level Forecasting
* Trend Analysis
* Future Sales Estimation

### Reporting Automation

* Automated Excel Report Generation
* Forecast Workbooks
* Executive Summary Creation
* Email Notification System

### Database Support

* SQLite (Development)
* PostgreSQL (Production)

## Installation

### Clone Repository

cd RetailSalesAnalytics

### Create Virtual Environment

python -m venv venv

venv\Scripts\activate

### Install Dependencies

pip install -r requirements.txt

### Run Application

python python/main.py
